# 南哪寻宝（NJU LostFound）

南京大学失物招领与寻物启事平台。本仓库当前实现"发帖闭环"。

## 快速开始

```bash
docker compose up --build
# 前端：http://localhost:5173
# 后端：http://localhost:8000/api/health
```
